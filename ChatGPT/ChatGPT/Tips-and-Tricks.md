# ChatGPT Tips and Tricks

## Tip 1 — Be Specific

Instead of:

Write a story.

Use:

Write a 500-word sci-fi story for teenagers with an unexpected ending.

---

## Tip 2 — Assign a Role

Example:

You are a professional prompt engineer.

---

## Tip 3 — Give Context

Example:

Platform: YouTube

Audience: Beginners

Language: Hindi

Length: 60 seconds

---

## Tip 4 — Ask for Better Output

Examples:

- Make it more cinematic.
- Add more emotion.
- Improve the hook.
- Simplify the explanation.
- Add examples.

---

## Tip 5 — Request Multiple Versions

Example:

Generate 5 different hooks.

Generate 3 different titles.

---

## Tip 6 — Use Step-by-Step Prompts

Break large tasks into smaller parts.

Example:

Step 1: Research

Step 2: Outline

Step 3: Script

Step 4: Improve

---

## Tip 7 — Continue the Conversation

Ask:

- Continue
- Expand this section
- Add more examples
- Rewrite professionally

---

## Tip 8 — Choose an Output Format

Examples:

- Markdown
- JSON
- Table
- Bullet List
- CSV

---

## Tip 9 — Iterate

The first response is rarely the final one.

Improve the prompt and regenerate.

---

## Tip 10 — Save Good Prompts

Build your own prompt library for future projects.
