# ChatGPT Frequently Asked Questions (FAQ)

## What is ChatGPT?

ChatGPT is an AI assistant that helps with writing, coding, research, learning, brainstorming, and many other tasks using natural language.

---

## What can ChatGPT do?

- Answer questions
- Write articles
- Generate code
- Explain concepts
- Create scripts
- Translate languages
- Summarize documents
- Generate prompts
- Help with learning

---

## Can ChatGPT browse the internet?

Some versions and features support web browsing, while others work only with their built-in knowledge. Check the features available in your ChatGPT plan.

---

## Is ChatGPT always correct?

No.

Always verify important information, especially for legal, medical, financial, or rapidly changing topics.

---

## How do I write better prompts?

- Be specific.
- Give context.
- Assign a role.
- Mention the desired output format.
- Include examples when helpful.

---

## Can ChatGPT generate code?

Yes.

It can write, explain, debug, and improve code in many programming languages.

---

## Can ChatGPT create images?

Yes, in versions that support image generation or editing.

---

## Can ChatGPT help with studying?

Yes.

It can explain concepts, create quizzes, summarize notes, and help with practice questions.

---

## What are Custom GPTs?

Custom GPTs are personalized AI assistants designed for specific tasks with custom instructions and workflows.

---

## Best Practices

- Ask clear questions.
- Break large tasks into smaller steps.
- Review AI-generated output.
- Save useful prompts.
- Keep learning and experimenting.

---

## Conclusion

The more clearly you communicate your goal, the better ChatGPT can help you.
