# ChatGPT Workflow Guide

## Step 1 — Define Your Goal

Decide exactly what you want ChatGPT to do.

Examples:
- Write a script
- Generate prompts
- Create code
- Research a topic
- Translate text

---

## Step 2 — Assign a Role

Examples:

- Professional Script Writer
- Prompt Engineer
- AI Researcher
- Marketing Expert
- Software Developer
- Teacher

---

## Step 3 — Add Context

Provide enough information.

Example:

Topic: AI Video Generation

Audience: Beginners

Platform: YouTube

Language: Hindi

Duration: 60 seconds

---

## Step 4 — Set Requirements

Examples:

- Cinematic
- Detailed
- SEO Optimized
- Professional
- Beginner Friendly
- Realistic

---

## Step 5 — Choose Output Format

Examples:

- Markdown
- JSON
- Table
- Bullet List
- Code
- Step-by-Step Guide

---

## Step 6 — Review

Check the output.

Improve the prompt if needed.

Repeat until satisfied.

---
