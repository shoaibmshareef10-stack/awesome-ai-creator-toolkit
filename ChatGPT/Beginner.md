# ChatGPT Beginner Guide

## Introduction

ChatGPT is an AI assistant that helps with writing, coding, research, brainstorming, learning, prompt engineering, and content creation.

## What is ChatGPT?

ChatGPT is a conversational AI developed by OpenAI. It understands natural language and helps generate text, ideas, code, images (where supported), and creative content.

## Key Features

- Writing Assistance
- Coding Help
- Prompt Engineering
- Research
- Content Creation
- Marketing
- Brainstorming
- Translation
- Summarization

## Requirements

- ChatGPT account
- Internet connection
- Clear prompts

## Getting Started

1. Open ChatGPT.
2. Start a new chat.
3. Describe your goal.
4. Add context.
5. Review and refine the response.

## Basic Prompt Formula

Role + Goal + Context + Requirements + Output Format

### Example

Role: Professional YouTube Script Writer

Goal: Create a 60-second YouTube Shorts script.

Context: Topic is AI tools.

Output: Hindi language with hook, body, and CTA.
