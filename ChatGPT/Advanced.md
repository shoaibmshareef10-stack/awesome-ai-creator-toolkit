# Advanced Prompting Techniques

## Role Prompting

Assign ChatGPT a professional role.

Examples:

- Senior Software Engineer
- Prompt Engineer
- Marketing Expert
- YouTube Script Writer
- AI Researcher

---

## Chain of Thought

Ask ChatGPT to think step by step before answering.

Example:

Think step by step before generating the final answer.

# Prompt Optimization Tips

- Be specific.
- Add context.
- Define the audience.
- Mention tone.
- Specify format.
- Request revisions when needed.

# Best Practices

- Give clear instructions.
- Avoid vague prompts.
- Ask follow-up questions.
- Use Markdown formatting.
- Save successful prompts.
- Iterate until satisfied.

---

# Common Mistakes

- Prompts are too short.
- Missing context.
- Too many unrelated tasks.
- No output format.
