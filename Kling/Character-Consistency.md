# Kling Character Consistency

## Introduction

Character consistency ensures the same person or character looks identical across multiple scenes and videos.

---

# Character Description

Always describe:

- Face
- Hairstyle
- Eye Color
- Skin Tone
- Clothing
- Accessories
- Age
- Height
- Expression

---

# Example Character

Young man, short black hair, brown eyes, fair skin, black hoodie, blue jeans, white sneakers, smiling, cinematic style.

---

# Keep These Consistent

- Face shape
- Hairstyle
- Clothes
- Colors
- Body type
- Accessories
- Lighting style

---

# Scene Consistency

Even when changing locations:

- Keep the same character.
- Keep the same outfit unless intentionally changed.
- Maintain facial features.
- Use similar lighting when possible.

---

# Good Prompt Example

A young man with short black hair, black hoodie, blue jeans and white sneakers walking through a futuristic city, cinematic lighting, tracking shot, realistic, same appearance as previous scene.

---

# Common Mistakes

- Changing clothes every scene.
- Different hairstyle.
- Different face shape.
- Different body proportions.
- Different age.

---

# Pro Tips

- Save your best character prompt.
- Copy the same character description into every scene.
- Change only the environment or action.
- Test short scenes before creating long videos.
