# Kling Checklist

## Before Writing the Prompt

- [ ] Define the goal of the video.
- [ ] Choose the main subject.
- [ ] Decide the action.
- [ ] Select the environment.
- [ ] Choose the time of day.
- [ ] Add weather if needed.

---

## Character Checklist

- [ ] Same face.
- [ ] Same hairstyle.
- [ ] Same clothing.
- [ ] Same accessories.
- [ ] Same body proportions.
- [ ] Same expression when required.

---

## Camera Checklist

- [ ] Camera angle selected.
- [ ] Camera movement selected.
- [ ] Smooth motion.
- [ ] Subject stays in frame.

---

## Lighting Checklist

- [ ] Cinematic lighting.
- [ ] Correct light direction.
- [ ] Shadows look natural.
- [ ] Lighting matches the mood.

---

## Environment Checklist

- [ ] Detailed background.
- [ ] Foreground elements.
- [ ] Realistic atmosphere.
- [ ] Good depth.
- [ ] No unnecessary objects.

---

## Quality Checklist

- [ ] High-quality prompt.
- [ ] Clear description.
- [ ] No conflicting instructions.
- [ ] Natural movement.
- [ ] Realistic style.

---

## Final Review

Before generating the video, confirm:

- [ ] Prompt is complete.
- [ ] Character is consistent.
- [ ] Camera movement is correct.
- [ ] Lighting is cinematic.
- [ ] Environment is realistic.
- [ ] Video matches the intended story.

---
