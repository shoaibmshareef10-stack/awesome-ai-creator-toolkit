# ⚡ Code Generation

## Overview

Cursor can generate code from natural language prompts, helping you build applications faster and reduce repetitive work.

---

## What Can Cursor Generate?

- Functions
- Classes
- APIs
- Websites
- Mobile Apps
- Database Queries
- Unit Tests
- Documentation
- Scripts
- Configuration Files

---

## Example Prompts

### Create a Function

```
Create a Python function to calculate the factorial of a number.
```

### Build an API

```
Create a REST API using Express.js with CRUD operations for a Todo app.
```

### Generate HTML

```
Create a responsive landing page using HTML, CSS, and JavaScript.
```

### Create SQL

```
Write an SQL query to find the top 10 highest-paid employees.
```

### Write Unit Tests

```
Generate unit tests for this JavaScript function using Jest.
```

---

## Best Practices

- Clearly describe your requirements.
- Mention the programming language.
- Include the desired framework.
- Request comments if needed.
- Test generated code before deployment.

---

## Tips

- Generate small components first.
- Improve results with follow-up prompts.
- Refactor generated code when necessary.
- Use version control before making large changes.
