# ⌨️ Keyboard Shortcuts

## Overview

Keyboard shortcuts help you navigate Cursor faster, improve productivity, and reduce reliance on the mouse.

---

## Essential Shortcuts

| Action | Windows/Linux | macOS |
|---------|---------------|--------|
| Command Palette | Ctrl + Shift + P | Cmd + Shift + P |
| Quick Open | Ctrl + P | Cmd + P |
| Search | Ctrl + Shift + F | Cmd + Shift + F |
| Save File | Ctrl + S | Cmd + S |
| New File | Ctrl + N | Cmd + N |
| Open Folder | Ctrl + K Ctrl + O | Cmd + K Cmd + O |
| Terminal | Ctrl + ` | Cmd + ` |
| Toggle Sidebar | Ctrl + B | Cmd + B |

## AI Shortcuts

Use keyboard shortcuts to:

- Open AI Chat
- Generate Code
- Explain Selected Code
- Fix Errors
- Refactor Code

> Shortcut keys may change between Cursor versions. Check **Settings → Keyboard Shortcuts** for the latest mappings.

---

## Navigation Tips

- Learn the Command Palette.
- Use Quick Open to jump between files.
- Search across your project instead of browsing folders.
- Customize shortcuts to match your workflow.

---

## Best Practices

- Memorize the shortcuts you use daily.
- Keep frequently used commands on easy key combinations.
- Review new shortcuts after major Cursor updates.
