# 🎼 Composer

## Overview

Composer is one of Cursor's most powerful AI features. It can understand your project, plan changes, and generate or edit multiple files from a single prompt.

---

## What Can Composer Do?

- Create complete projects
- Edit multiple files at once
- Add new features
- Refactor code
- Fix bugs across a project
- Generate documentation
- Update existing code

---

## How Composer Works

1. Open your project.
2. Launch Composer.
3. Describe what you want in natural language.
4. Review the AI-generated plan.
5. Accept or modify the proposed changes.

---

## Example Prompts

### Build a Feature

```
Add user authentication with Firebase to this project.
```

### Create a Dashboard

```
Build a responsive admin dashboard with charts and dark mode.
```

### Refactor a Project

```
Refactor this project using reusable React components.
```

### Improve Documentation

```
Generate documentation for every public function.
```

---

## Best Practices

- Be specific about your requirements.
- Mention the framework and programming language.
- Review all generated changes before accepting them.
- Commit your work before making large AI changes.

---

## Tips

- Start with small tasks.
- Use follow-up prompts to refine results.
- Review each generated file carefully.
- Test the project after applying changes.
