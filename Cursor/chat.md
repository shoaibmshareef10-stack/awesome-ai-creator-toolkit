# 💬 AI Chat

## Overview

Cursor AI Chat helps you interact with your codebase using natural language. You can ask questions, generate code, explain functions, and fix bugs without leaving the editor.

---

## What You Can Do

- Ask coding questions
- Explain existing code
- Generate new code
- Fix bugs
- Refactor code
- Write documentation
- Create unit tests

---

## Example Prompts

### Explain Code

```
Explain what this function does.
```

### Generate Code

```
Create a Python function to sort a list of dictionaries by age.
```

### Debug Code

```
Find the bug in this code and explain how to fix it.
```

### Optimize Code

```
Optimize this function for better performance.
```

### Add Comments

```
Add clear comments to this code.
```

---

## Best Practices

- Write clear and specific prompts.
- Provide enough context.
- Review AI-generated code before using it.
- Test all generated code.

---

## Tips

- Break complex tasks into smaller prompts.
- Use follow-up questions to improve results.
- Keep your project files organized for better AI context.
