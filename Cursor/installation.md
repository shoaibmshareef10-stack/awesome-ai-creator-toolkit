# 💻 Installing Cursor

## System Requirements

- Windows 10/11
- macOS
- Linux
- Internet connection

---

## Download Cursor

Visit the official Cursor website and download the version for your operating system.

---

## Installation

### Windows

1. Download the installer.
2. Run the installer.
3. Follow the setup wizard.
4. Launch Cursor.

### macOS

1. Download the DMG file.
2. Drag Cursor into the Applications folder.
3. Open Cursor.

### Linux

Install the package for your Linux distribution and launch Cursor.

---

## Sign In

Sign in with your preferred account to unlock AI features.

---

## Verify Installation

Open Cursor and create a new project. If the AI chat opens successfully, your installation is complete.
