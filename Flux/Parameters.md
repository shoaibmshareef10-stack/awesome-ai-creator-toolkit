# Flux Parameters Guide

## Introduction

Parameters allow you to control the quality, composition, consistency, and creativity of images generated with Flux. Learning these settings helps you produce more predictable and professional results.

---

# Aspect Ratio

Choose the correct image size for your project.

Common aspect ratios:

- 1:1 (Square)
- 3:4 (Portrait)
- 4:3 (Landscape)
- 9:16 (Vertical)
- 16:9 (Widescreen)

---

# Prompt Strength

Controls how closely Flux follows your prompt.

- Low: More creative freedom
- Medium: Balanced results
- High: Strict prompt accuracy

---

# Guidance Scale

Determines how strongly the AI follows your instructions.

Recommended:

- 6–8 for creative work
- 8–10 for accurate prompt following

---

# Seed

A seed lets you reproduce similar images.

Use the same seed when creating consistent characters or scenes.

---

# Number of Images

Generate multiple images to compare results.

Recommended:

- 4 images for testing
- 8 images for professional selection

---

# Image Quality

Use higher quality settings for final outputs.

Lower quality is useful for quick previews.

---

# Recommended Workflow

1. Write a detailed prompt.
2. Select the aspect ratio.
3. Adjust the guidance scale.
4. Choose the image quality.
5. Generate multiple variations.
6. Save the best seed.
7. Upscale the final image.

---

# Best Practices

- Keep parameters consistent for similar projects.
- Save successful settings.
- Test one parameter at a time.
- Experiment to find your preferred workflow.
