# Flux Checklist

## Before Writing the Prompt

- [ ] Define the main subject.
- [ ] Describe the action or pose.
- [ ] Choose the environment.
- [ ] Select the art style.
- [ ] Decide the mood.
- [ ] Add the lighting.
- [ ] Choose the camera angle.
- [ ] Select the aspect ratio.
- [ ] Adjust the guidance scale.
- [ ] Decide whether to use a seed.

---

# Prompt Structure

Subject → Action → Environment → Lighting → Camera Angle → Style → Mood → Quality → Parameters

---

# Image Quality Checklist

- [ ] Subject is clearly described.
- [ ] Background matches the scene.
- [ ] Lighting is realistic.
- [ ] Camera angle supports the composition.
- [ ] Style is consistent.
- [ ] Colors are balanced.
- [ ] Image is sharp and detailed.
- [ ] Aspect ratio is correct.

---

# Parameter Checklist

- [ ] Guidance Scale is optimized.
- [ ] Prompt Strength is appropriate.
- [ ] Seed is saved (if needed).
- [ ] Image quality is set correctly.
- [ ] Multiple variations are generated.

---

# Final Review

Before generating the image, confirm:

- [ ] Prompt is complete.
- [ ] No conflicting keywords.
- [ ] Lighting matches the mood.
- [ ] Camera angle fits the subject.
- [ ] Style is consistent.
- [ ] Parameters are ready.
