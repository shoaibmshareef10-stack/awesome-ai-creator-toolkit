## AI Workflow

1. Describe your requirement.
2. Generate the initial solution.
3. Review the generated output.
4. Refine the prompt if needed.
5. Test the application.
6. Repeat until satisfied.
7. Deploy the project.

## Best Practices

- Write clear prompts.
- Review AI-generated code carefully.
- Test every generated feature.
- Make small, incremental changes.
- Keep project requirements specific.

---

## Common Mistakes

- Expecting perfect output on the first try.
- Using vague instructions.
- Skipping testing.
- Ignoring security considerations.
- Deploying without reviewing the generated code.
