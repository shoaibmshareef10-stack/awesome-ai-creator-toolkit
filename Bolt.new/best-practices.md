# ⭐ Best Practices

## Overview

Following best practices in Bolt.new helps you create reliable, maintainable, and scalable applications. Combining clear AI prompts with proper development workflows leads to better project quality and fewer errors.

---

## Plan Before You Build

Before generating code:

- Define the project goal.
- Identify core features.
- Choose the required technologies.
- Break large projects into smaller tasks.
- Create a clear development plan.

---

## Write Better Prompts

To get high-quality results:

- Be specific.
- Describe one feature at a time.
- Include important requirements.
- Mention preferred technologies when necessary.
- Refine prompts based on previous results.

---

## Review AI-Generated Code

Always:

- Read generated code carefully.
- Verify business logic.
- Check for security issues.
- Remove unnecessary code.
- Improve readability where needed.

---

## Test Frequently

Regular testing helps you:

- Detect bugs early.
- Validate new features.
- Prevent regressions.
- Improve application stability.
- Deliver a better user experience.

---

## Organize Your Project

Maintain a clean project by:

- Using meaningful file names.
- Grouping related components.
- Removing unused files.
- Keeping folders organized.
- Following consistent naming conventions.

---

## Secure Your Application

Protect your project by:

- Storing secrets in environment variables.
- Never exposing API keys.
- Validating user input.
- Applying authentication where required.
- Keeping dependencies updated.

---

## Optimize Performance

Improve performance by:

- Reducing unnecessary code.
- Optimizing assets.
- Loading resources efficiently.
- Minimizing network requests.
- Monitoring application performance.

---

## Common Mistakes

Avoid:

- Writing vague prompts.
- Skipping code reviews.
- Ignoring testing.
- Deploying unfinished features.
- Leaving sensitive information in the codebase.

---

## Summary

Applying these best practices helps you build secure, efficient, and maintainable Bolt.new applications while making the most of AI-assisted development.
