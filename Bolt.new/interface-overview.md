# 🖥️ Interface Overview

## Overview

The Bolt.new interface is designed to help you build, edit, preview, and deploy full-stack applications using AI. Its browser-based workspace combines code generation, live preview, and project management in a single environment.

---

## Main Components

The Bolt.new interface includes:

- Dashboard
- Project Workspace
- AI Prompt Panel
- Code Editor
- Live Preview
- File Explorer
- Terminal
- Deployment Panel

---

## Dashboard

The dashboard allows you to:

- Create new projects
- Open existing projects
- Manage project settings
- Access recent work
- View deployment status

---

## Project Workspace

The Project Workspace is where you build and manage your application.

You can:

- Write AI prompts
- Generate code
- Edit project files
- Test your application
- Save project changes

---

## Code Editor

The Code Editor enables you to:

- View generated code
- Edit files manually
- Organize project structure
- Search project files
- Debug your application

---

## Live Preview

The Live Preview allows you to:

- View application changes instantly
- Test user interactions
- Verify layouts
- Identify UI issues
- Validate generated features

---

## Deployment Panel

The Deployment Panel helps you:

- Prepare your application
- Deploy projects
- Monitor deployment status
- Manage production versions

---

## Best Practices

- Organize project files clearly.
- Write specific AI prompts.
- Review generated code regularly.
- Test changes before deployment.
- Save project versions frequently.

---

## Common Mistakes

- Using vague prompts.
- Ignoring generated code.
- Skipping application testing.
- Making large changes without verification.
- Deploying untested projects.

---

## Summary

Understanding the Bolt.new interface helps you work more efficiently, build applications faster, and manage AI-assisted development with confidence.
