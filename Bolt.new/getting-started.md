# 🚀 Getting Started

## Overview

Bolt.new is an AI-powered development platform that enables you to create full-stack web applications using natural language prompts. It combines AI code generation with an online development environment to help you build applications quickly.

---

## What is Bolt.new?

Bolt.new helps you:

- Build applications with AI
- Generate code automatically
- Edit projects using prompts
- Prototype ideas quickly
- Deploy applications efficiently

---

## Key Concepts

Before building projects, understand these concepts:

- AI Prompts
- Projects
- Workspace
- Code Generation
- Live Preview
- Integrations
- Deployment

---

## First Steps

1. Create a Bolt.new account.
2. Open the workspace.
3. Create a new project.
4. Write your first prompt.
5. Review the generated code.
6. Test the application.
7. Save or deploy your project.

## Best Practices

- Start with simple prompts.
- Build features step by step.
- Test generated code regularly.
- Keep prompts clear and specific.
- Save project versions frequently.

---

## Common Mistakes

- Writing vague prompts.
- Skipping code review.
- Building everything in one prompt.
- Ignoring testing.
- Forgetting to save project changes.

---

## Summary

Bolt.new simplifies application development by combining AI-assisted coding with a browser-based development environment, enabling developers to build projects faster and more efficiently.
