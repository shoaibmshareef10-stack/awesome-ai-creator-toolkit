# 💻 Installation

## Overview

Before building applications with Bolt.new, you need to create an account, configure your workspace, and understand the basic project setup process.

---

## Requirements

You'll need:

- A modern web browser
- A valid email address
- Internet connection
- Basic knowledge of web development (recommended)

---

## Account Setup

To get started:

1. Sign up for a Bolt.new account.
2. Verify your email address.
3. Log in to your workspace.
4. Complete your profile.
5. Explore the dashboard.

---

## Workspace Configuration

After signing in:

- Create your first project.
- Configure project settings.
- Explore the AI prompt editor.
- Open the live preview.
- Review project preferences.

---

## Creating Your First Project

Before generating code:

- Choose a project name.
- Describe your application with a clear prompt.
- Review the generated project.
- Test the application.
- Save your changes.

---

## Verify Installation

Ensure that:

- Your account is active.
- Your workspace loads successfully.
- AI code generation works.
- Live preview is available.
- You can create a new project.

---

## Best Practices

- Use a secure password.
- Enable two-factor authentication (2FA) if available.
- Keep projects organized.
- Save your work frequently.
- Review generated code before deployment.

---

## Common Mistakes

- Writing unclear prompts.
- Skipping project testing.
- Ignoring generated errors.
- Not saving project versions.
- Deploying without verification.

---

## Summary

A properly configured Bolt.new workspace provides a strong foundation for AI-assisted full-stack development and rapid application creation.
