## Prompt Template

Role:
You are an AI automation assistant.

Task:
Automate the following workflow.

Input:
{{User Input}}

Expected Output:
Step-by-step workflow with required nodes.

---

## Tips

- Use clear prompts
- Break large tasks into smaller workflows
- Test every node
- Use error handling
- Reuse workflows whenever possible
