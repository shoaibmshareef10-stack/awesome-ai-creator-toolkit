# 🤖 n8n Beginner Guide

## What is n8n?

n8n is an open-source workflow automation platform that connects apps, APIs, AI models, and databases without writing much code.

---

## Features

- Visual Workflow Builder
- AI Agent Support
- API Integration
- Webhooks
- Scheduling
- Database Integration
- Self-hosting

---

## Getting Started

1. Install n8n
2. Create your account
3. Build your first workflow
4. Add nodes
5. Connect applications
6. Test and activate

---

## Common Nodes

- HTTP Request
- Webhook
- OpenAI
- Gmail
- Google Sheets
- Slack
- Telegram

## Example Workflow

Webhook → AI → Google Sheets → Gmail Notification
