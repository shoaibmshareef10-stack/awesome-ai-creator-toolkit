# Veo Character Consistency Guide

## Keep Character Consistent

- Same face
- Same hairstyle
- Same skin tone
- Same eye color
- Same body type
- Same outfit
- Same accessories
- Same voice
- Same age
- Same personality

## Prompt Example

The same young man with short black hair, brown eyes, medium skin tone, wearing a black leather jacket, blue jeans, white sneakers, maintaining identical facial features, body proportions, hairstyle, outfit, and expressions across every scene.
