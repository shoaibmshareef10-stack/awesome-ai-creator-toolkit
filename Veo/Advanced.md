# Negative Prompt

Avoid:

- blurry
- low quality
- extra fingers
- bad anatomy
- duplicate body parts
- watermark
- logo
- text
- artifacts
- noise

# Pro Tips

- Keep prompts detailed but clear.
- Maintain character consistency.
- Match lighting with the environment.
- Use cinematic camera movement.
- Describe emotions naturally.
- End every prompt with:
  **Ultra realistic, cinematic, HDR, highly detailed, 8K.**
