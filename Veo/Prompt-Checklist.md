# Veo Prompt Checklist

## Introduction

Use this checklist before generating any video to ensure your prompt is complete, consistent, and optimized for Veo.

---

# Character

- [ ] Main subject clearly described
- [ ] Face details included
- [ ] Hairstyle mentioned
- [ ] Outfit specified
- [ ] Accessories included
- [ ] Facial expression defined
- [ ] Body language described

---

# Action

- [ ] Main action is clear
- [ ] Motion is realistic
- [ ] Emotion matches the action

---

# Environment

- [ ] Location defined
- [ ] Time of day included
- [ ] Weather described
- [ ] Background details added
- [ ] Atmosphere explained

---

# Camera

- [ ] Camera angle selected
- [ ] Camera movement specified
- [ ] Lens mentioned
- [ ] Camera speed defined

---

# Lighting

- [ ] Light source selected
- [ ] Direction mentioned
- [ ] Shadow style included
- [ ] Mood matches the scene

---

# Visual Style

- [ ] Art style selected
- [ ] Color palette defined
- [ ] Cinematic look specified

Examples:

- Hollywood
- Documentary
- Sci-Fi
- Cyberpunk
- Fantasy
- Realistic

---

# Quality

- [ ] Ultra Realistic
- [ ] HDR
- [ ] High Detail
- [ ] 4K / 8K
- [ ] Cinematic Composition

---

# Audio (Optional)

- [ ] Background music
- [ ] Ambient sounds
- [ ] Sound effects

---

# Final Review

Before generating:

- [ ] Character consistency maintained
- [ ] Environment consistency maintained
- [ ] Camera movement is realistic
- [ ] Lighting matches the story
- [ ] No conflicting instructions
- [ ] Prompt is clear and detailed

---

# Pro Tips

- Keep prompts specific.
- Avoid unnecessary repetition.
- Reuse successful prompt templates.
- Test multiple prompt variations.
- Save your best prompts for future projects.
