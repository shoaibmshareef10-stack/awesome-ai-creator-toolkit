# 🐘 PostgreSQL Database

## Overview

Supabase is built on PostgreSQL, a powerful open-source relational database. It provides structured data storage, advanced querying, strong consistency, and enterprise-grade reliability for modern applications.

---

## What is PostgreSQL?

PostgreSQL is a relational database management system (RDBMS) that stores data in tables connected by relationships.

It allows you to:

- Store structured data
- Create relationships between tables
- Execute SQL queries
- Manage transactions
- Scale applications reliably

---

## Database Components

A PostgreSQL database includes:

- Tables
- Rows
- Columns
- Primary Keys
- Foreign Keys
- Indexes
- Views

## Typical Workflow

1. Create database tables.
2. Define relationships.
3. Insert data.
4. Query records.
5. Update or delete data.
6. Optimize database performance.

## Benefits

- Relational data model
- ACID compliance
- Powerful SQL support
- High performance
- Automatic scalability

---

## Best Practices

- Design normalized tables.
- Use indexes wisely.
- Define primary and foreign keys.
- Optimize SQL queries.
- Back up your database regularly.

---

## Common Mistakes

- Poor database design.
- Missing indexes.
- Writing inefficient SQL queries.
- Ignoring relationships.
- Storing duplicate data.
