# ⚙️ Installation

## Overview

Before using Supabase services, you need to install the Supabase SDK and connect it to your application. A proper installation ensures secure communication with your Supabase project.

---

## Requirements

Make sure you have:

- Node.js (latest LTS version recommended)
- npm or another package manager
- A Supabase account
- A Supabase project
- A code editor

---

## Install Supabase SDK

Install the JavaScript client using npm:

```bash
npm install @supabase/supabase-js
```

Or using Yarn:

```bash
yarn add @supabase/supabase-js
```

Or using pnpm:

```bash
pnpm add @supabase/supabase-js
```

---

## Verify Installation

Confirm the package is installed:

```bash
npm list @supabase/supabase-js
```

---

## Connect Your Application

After installation:

1. Create a Supabase project.
2. Obtain the Project URL.
3. Obtain the API Key.
4. Initialize the Supabase client.
5. Connect the required services.

---

## Recommended Development Tools

- Visual Studio Code
- Git
- GitHub
- Node.js
- Supabase CLI

---

## Installation Tips

- Use the latest SDK version.
- Keep dependencies updated.
- Store API keys securely.
- Test the connection before development.

---

## Common Issues

- Node.js not installed.
- Invalid Project URL.
- Incorrect API key.
- Missing SDK installation.
- Internet connectivity issues.

---

## Summary

Installing the Supabase SDK is the first step toward building secure and scalable applications using Supabase's backend services.
