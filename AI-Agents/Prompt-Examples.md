# 🤖 AI Agents - Prompt Examples

## Example 1: Research Agent

**Prompt**

Research the latest AI video generation tools. Compare their features, pricing, advantages, disadvantages, and recommend the best option for beginners.

---

## Example 2: Coding Agent

**Prompt**

Build a responsive landing page using HTML, CSS, and JavaScript. Include comments, modern UI, and SEO best practices.

---

## Example 3: Content Creation Agent

**Prompt**

Write a 60-second YouTube Shorts script with a strong hook, storytelling, CTA, title, description, and hashtags.

---

## Example 4: Marketing Agent

**Prompt**

Create a complete Instagram growth strategy for an AI creator from 0 to 100,000 followers.

---

## Example 5: Business Agent

**Prompt**

Generate a business plan for an AI automation agency including pricing, services, target audience, and marketing strategy.

---

## Tips

- Be specific.
- Define the desired output.
- Mention the format.
- Provide context.
- Iterate to improve results.
