# Leonardo Checklist

## Before Writing the Prompt

- [ ] Define the main subject.
- [ ] Describe the action or pose.
- [ ] Select the environment.
- [ ] Choose the artistic style.
- [ ] Decide the mood.
- [ ] Add lighting.
- [ ] Select the camera angle.
- [ ] Choose the aspect ratio.
- [ ] Set the guidance scale.
- [ ] Decide whether to use a seed.

---

# Prompt Structure

Subject → Action → Environment → Lighting → Camera → Style → Mood → Quality → Parameters

---

# Image Quality Checklist

- [ ] Subject is clearly described.
- [ ] Background matches the subject.
- [ ] Lighting is realistic.
- [ ] Camera angle is appropriate.
- [ ] Art style is consistent.
- [ ] Colors are balanced.
- [ ] Image composition is clean.
- [ ] Aspect ratio is correct.

---

# Parameter Checklist

- [ ] Guidance Scale is optimized.
- [ ] Prompt Strength is suitable.
- [ ] Seed is saved (if consistency is needed).
- [ ] Image quality is set correctly.
- [ ] Number of images is appropriate.

---

# Final Review

Before generating the image, confirm:

- [ ] Prompt is complete.
- [ ] No conflicting keywords.
- [ ] Style matches the project.
- [ ] Lighting enhances the mood.
- [ ] Camera angle supports the composition.
- [ ] Parameters are ready.
