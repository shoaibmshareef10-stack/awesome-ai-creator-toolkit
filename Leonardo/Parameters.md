# Leonardo Parameters Guide

## Introduction

Parameters help control the appearance, quality, composition, and consistency of images generated with Leonardo AI.

---

# Image Size

Choose the right aspect ratio for your project.

Common ratios:

- 1:1 (Square)
- 3:4 (Portrait)
- 4:3 (Landscape)
- 9:16 (Vertical)
- 16:9 (Wide)

---

# Prompt Strength

Controls how closely the AI follows your prompt.

- Low = More creative
- Medium = Balanced
- High = Follows the prompt closely

---

# Guidance Scale

Higher values make the AI follow your prompt more accurately.

Recommended:

- 7–10 for most images.

---

# Seed

A seed helps reproduce similar images.

Use the same seed to generate consistent results.

---

# Number of Images

Generate multiple images to compare results.

Recommended:

- 4 images for exploration.
- 8 images for professional work.

---

# Image Quality

Choose higher quality for final renders.

Lower quality is useful for quick testing.

---

# Example Workflow

1. Write a detailed prompt.
2. Select aspect ratio.
3. Set guidance scale.
4. Generate multiple images.
5. Compare results.
6. Upscale the best image.

---

# Best Practices

- Keep settings consistent for similar projects.
- Save successful parameter combinations.
- Experiment with different values.
- Use the same seed for character consistency.

---

# Common Mistakes

- Changing too many settings at once.
- Using random aspect ratios.
- Ignoring image quality.
- Not saving successful settings.
