# Leonardo Advanced Guide

## Introduction

This guide covers advanced techniques for creating professional AI images using Leonardo AI.

---

# Advanced Prompt Formula

Subject + Action + Environment + Lighting + Camera + Style + Mood + Quality + Parameters

---

# Character Consistency

- Keep the same face.
- Use identical clothing.
- Maintain hairstyle.
- Keep colors consistent.

---

- Drone View

---

# Lighting

- Cinematic Lighting

- Volumetric Lighting

---

# Professional Workflow

1. Define your goal.
2. Write a detailed prompt.
3. Choose the best AI model.
4. Add lighting.
5. Select camera angle.
6. Generate multiple images.
7. Upscale the best result.
8. Refine if needed.

---

# Pro Tips

- Use detailed prompts.
- Mention camera angle.
- Add lighting.
- Keep prompts organized.
- Save successful prompts.

---

# Common Mistakes

- Very short prompts.
- Conflicting styles.
- Missing lighting.
- Too many unrelated keywords.
