# Midjourney Parameters Guide

## Introduction

Parameters control how Midjourney generates images. They help customize aspect ratio, creativity, quality, speed, and consistency.

## Essential Parameters

### --ar (Aspect Ratio)

Controls image dimensions.

Examples:
- --ar 1:1
- --ar 16:9
- --ar 9:16
- --ar 4:5

---

### --stylize

Controls artistic creativity.

Examples:
- --stylize 50
- --stylize 100
- --stylize 500

---

### --chaos

Controls randomness.

Examples:
- --chaos 0
- --chaos 25
- --chaos 50
- --chaos 100

---

### --seed

Keeps results more consistent.

Example:

--seed 12345

---

### --quality

Controls rendering quality.

Examples:

- --quality 0.5
- --quality 1
- --quality 2

---

### --weird

Creates unusual artistic results.

Example:

--weird 200

---

## Example Prompt

Futuristic superhero, cinematic lighting, 85mm lens, ultra realistic --ar 9:16 --stylize 150 --chaos 20 --quality 2

## Best Practices

- Use only necessary parameters.
- Keep the aspect ratio suitable for your platform.
- Save successful parameter combinations.
- Experiment one parameter at a time.

## Common Mistakes

- Using too many parameters together.
- Very high chaos for consistent characters.
- Wrong aspect ratio for the intended use.
