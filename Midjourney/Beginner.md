# Midjourney Beginner Guide

## What is Midjourney?

Midjourney is an AI image generation tool that creates high-quality artwork, realistic photos, illustrations, and concept designs from text prompts.

## Getting Started

1. Create a Midjourney account.
2. Join the Midjourney server or use the web app.
3. Type a clear prompt.
4. Generate the image.
5. Upscale or create variations.

## Basic Prompt Structure

Subject + Environment + Lighting + Camera + Style + Quality

### Example

A futuristic city at sunset, cinematic lighting, ultra realistic, 8K, highly detailed.

## Tips

- Use clear and descriptive prompts.
- Mention lighting and camera angle.
- Add art style if needed.
- Keep prompts simple at first.
- Experiment with different versions.

## Common Mistakes

- Very short prompts.
- Too many unrelated keywords.
- No lighting or style.
- Unrealistic expectations from a single prompt.
