# Midjourney Advanced Guide

## Introduction

This guide covers advanced prompting techniques to create professional-quality AI images with Midjourney.

## Advanced Prompt Formula

Subject + Action + Environment + Lighting + Camera + Composition + Style + Mood + Quality + Parameters

## Important Parameters

### --ar
Changes the aspect ratio.

Examples:
- --ar 1:1
- --ar 16:9
- --ar 9:16

### --stylize

Controls artistic creativity.

Example:
--stylize 100

### --chaos

Adds randomness.

Example:
--chaos 30

### --seed

Keeps image generation consistent.

Example:
--seed 12345

## Character Consistency

- Use the same prompt.
- Use the same seed.
- Keep clothing and hairstyle identical.
- Maintain the same camera angle.

## Lighting Styles

- Cinematic Lighting
- Golden Hour
- Soft Light
- Studio Lighting
- Neon Lighting
- Volumetric Lighting

## Camera Styles

- Close-up
- Wide Shot
- Low Angle
- High Angle
- Drone View
- Eye Level

## Professional Workflow

1. Write a clear prompt.
2. Choose the aspect ratio.
3. Add lighting.
4. Add camera angle.
5. Add parameters.
6. Generate multiple versions.
7. Upscale the best image.

## Pro Tips

- Keep prompts specific.
- Use reference images when needed.
- Save your best prompts.
- Test different parameters.

## Common Mistakes

- Overloading prompts.
- Using random keywords.
- Ignoring lighting.
- Changing multiple settings at once.
