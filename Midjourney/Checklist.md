# Midjourney Checklist

## Before You Generate

- [ ] Subject is clearly defined.
- [ ] Environment is described.
- [ ] Lighting is selected.
- [ ] Camera angle is chosen.
- [ ] Art style is added.
- [ ] Mood is specified.
- [ ] Quality keywords are included.
- [ ] Correct aspect ratio is selected.
- [ ] Parameters are reviewed.

---

## Prompt Formula

Subject → Action → Environment → Lighting → Camera → Style → Mood → Quality → Parameters

## Final Review

- [ ] Prompt is complete.
- [ ] No conflicting keywords.
- [ ] Character consistency maintained.
- [ ] Composition looks balanced.
- [ ] Parameters are correct.

Generate multiple versions and choose the best result before upscaling.
