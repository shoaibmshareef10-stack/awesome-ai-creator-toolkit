# Code of Conduct

## Our Pledge

We are committed to creating a welcoming, respectful, and inclusive community for everyone.

## Our Standards

Examples of positive behavior:

- Be respectful and constructive.
- Help others learn and grow.
- Welcome new contributors.
- Accept feedback gracefully.

Examples of unacceptable behavior:

- Harassment or discrimination.
- Personal attacks.
- Offensive language.
- Spam or malicious activity.

## Enforcement

Project maintainers may edit, remove, or reject contributions that violate this Code of Conduct.

## Contact

If you have any questions or concerns, please open a GitHub Issue.
