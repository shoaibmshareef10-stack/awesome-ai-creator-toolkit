# Nano Banana Beginner Guide

## Introduction

Nano Banana is an AI-powered image generation and editing tool designed to create high-quality visuals quickly and easily. It is ideal for content creators, designers, marketers, and beginners.

---

# Getting Started

To create better images:

- Describe the main subject clearly.
- Add the environment.
- Choose the lighting.
- Select the camera angle.
- Mention the art style.
- Include quality keywords.

---

# Basic Prompt Formula

Subject + Action + Environment + Lighting + Camera Angle + Style + Quality

---

# Example Prompt

A futuristic robot standing in a neon city at night, cinematic lighting, low-angle shot, ultra realistic, highly detailed, 8K.

---

# Popular Styles

- Realistic
- Cinematic
- Anime
- Fantasy
- Cyberpunk
- Pixar
- Digital Art
- Watercolor

---

# Tips

- Keep prompts simple and specific.
- Use realistic lighting.
- Focus on one subject.
- Experiment with different styles.

---

# Common Mistakes

- Very short prompts.
- Conflicting styles.
- Missing lighting details.
- Overloaded descriptions.

---

# Practice Prompt

A majestic white horse running across a green meadow during sunrise, golden hour lighting, cinematic composition, ultra realistic.
