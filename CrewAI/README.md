# 👥 CrewAI

## Overview

CrewAI is an open-source framework for building collaborative AI agent systems. Instead of using a single AI agent, CrewAI enables multiple specialized agents to work together on complex tasks through teamwork, planning, and communication.

It is designed for creating autonomous AI workflows such as research assistants, coding teams, content creators, customer support systems, and business automation.
