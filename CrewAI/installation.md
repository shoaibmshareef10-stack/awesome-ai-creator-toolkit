# ⚙️ Installation

## Overview

Before building AI agents with CrewAI, you need to install the required packages and configure your development environment.

---

## Requirements

Make sure you have:

- Python 3.10 or later
- pip
- A code editor
- API key for your preferred LLM provider

---

## Install CrewAI

Install CrewAI using pip:

```bash
pip install crewai
```

To install optional tools:

```bash
pip install 'crewai[tools]'
```

---

## Verify Installation

Check that CrewAI is installed correctly:

```bash
crewai --version
```

---

## Configure Environment

Store your API keys in environment variables instead of hardcoding them.

Common environment variables include:

- OPENAI_API_KEY
- ANTHROPIC_API_KEY
- GOOGLE_API_KEY

---

## Recommended Development Tools

- Visual Studio Code
- Git
- GitHub
- Python Virtual Environment

---

## Installation Tips

- Use a virtual environment for every project.
- Keep CrewAI updated.
- Store secrets securely.
- Test the installation before creating agents.

---

## Common Issues

- Python version is too old.
- Missing dependencies.
- Incorrect API key configuration.
- Virtual environment not activated.

---

## Summary

A proper installation ensures your CrewAI projects run smoothly and are ready for building multi-agent AI applications.
