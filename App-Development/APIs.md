# 🌐 APIs Guide

## What is an API?

An API (Application Programming Interface) allows two applications to communicate with each other.

---

## Types of APIs

- REST API
- GraphQL
- SOAP
- WebSocket

---

## Common HTTP Methods

- GET
- POST
- PUT
- PATCH
- DELETE

---

## Status Codes

- 200 OK
- 201 Created
- 400 Bad Request
- 401 Unauthorized
- 404 Not Found
- 500 Internal Server Error

---

## API Best Practices

- Use HTTPS
- Validate inputs
- Return proper status codes
- Version your APIs
- Write documentation

---

## Popular API Tools

- Postman
- Insomnia
- Swagger
- Bruno

---

## Learning Roadmap

HTTP Basics → REST APIs → JSON → Authentication → API Testing → API Documentation
