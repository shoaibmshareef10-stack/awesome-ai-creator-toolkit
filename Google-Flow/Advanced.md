# Professional Formula

[Subject] performing [Action] inside [Environment], cinematic camera movement, realistic lighting, detailed textures, volumetric fog, natural shadows, expressive facial emotions, physically accurate body movement, shallow depth of field, blockbuster composition, ultra realistic, 8K quality.

---

# Character Consistency

- Same hairstyle
- Same face
- Same clothes
- Same colors
- Same accessories
- Same age
- Same body proportions

Always describe the character exactly the same.

# Environment

Always describe:

- Weather
- Time of day
- Background
- Atmosphere
- Props
- Color palette

---

# Negative Prompt

Avoid:

- blurry
- low quality
- bad anatomy
- duplicate body parts
- extra fingers
- watermark
- text
- logo
- noise
- artifacts
- oversaturated colors

---

# Pro Tips

- Keep prompts specific.
- Use cinematic language.
- Maintain character consistency.
- Use realistic camera movement.
- Describe lighting clearly.
- Use detailed environments.
- End with quality tags:
  Ultra realistic, cinematic, blockbuster, 8K.
