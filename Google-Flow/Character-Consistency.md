# Character Consistency Guide

## Identity
- Keep the same face in every scene.
- Maintain consistent hairstyle.
- Preserve eye color and facial features.

## Outfit
- Use the same clothes unless the story changes.
- Keep colors and accessories consistent.

## Expressions
- Match emotions with the scene.
- Avoid random expression changes.

## Body
- Keep height and body proportions identical.
- Maintain posture consistency.

## Tips
- Use a character reference image.
- Repeat the same character description in every prompt.
- Lock identity before changing poses or camera angles.
