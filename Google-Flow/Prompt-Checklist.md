# Prompt Checklist

## Introduction

Before generating any AI video, review this checklist to ensure your prompt is complete, consistent, and cinematic.

---

# Subject

- [ ] Main subject clearly described
- [ ] Character appearance included
- [ ] Outfit specified
- [ ] Facial expression mentioned
- [ ] Body language included

---

# Action

- [ ] Clear action described
- [ ] Natural movement
- [ ] Emotion matches action

---

# Environment

- [ ] Location defined
- [ ] Time of day mentioned
- [ ] Weather included
- [ ] Background details added

---

# Camera

- [ ] Camera angle selected
- [ ] Camera movement described
- [ ] Lens style included (optional)

---

# Lighting

- [ ] Lighting type selected
- [ ] Shadow style described
- [ ] Mood matches lighting

---

# Style

- [ ] Visual style selected
- [ ] Color palette described
- [ ] Cinematic quality mentioned

Examples:
- Hollywood
- Pixar
- Anime
- Documentary
- Cyberpunk

---

# Quality

- [ ] 4K / 8K
- [ ] HDR
- [ ] Ultra realistic
- [ ] High detail
- [ ] Cinematic composition

---

# Audio (Optional)

- [ ] Background music
- [ ] Sound effects
- [ ] Ambient sounds

---

# Final Review

Before clicking Generate, confirm:

- [ ] No spelling mistakes
- [ ] Character consistency maintained
- [ ] Environment consistency maintained
- [ ] Camera movement is realistic
- [ ] Lighting matches the scene
- [ ] Prompt is clear and detailed

---

# Pro Tips

- Keep prompts simple but descriptive.
- Avoid conflicting instructions.
- Reuse consistent character descriptions.
- Test multiple prompt variations.
- Save successful prompts for future projects.
