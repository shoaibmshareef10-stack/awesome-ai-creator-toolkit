# Google Flow Beginner Guide

## Introduction

Google Flow is an AI filmmaking tool that helps create cinematic videos using text prompts, images, and reference assets.

## What is Google Flow?

Google Flow combines AI models to generate high-quality videos with realistic motion, lighting, camera movement, and character consistency.

## Features

- Text to Video
- Image to Video
- Character Consistency
- Environment Design
- Camera Controls
- Cinematic Lighting
- Prompt Templates
- JSON Workflow Support

## Requirements

- Google Account
- Google Flow Access
- Good Prompt Writing Skills
- Stable Internet Connection

## Getting Started

1. Sign in to Google Flow.
2. Create a new project.
3. Choose Text-to-Video or Image-to-Video.
4. Write a detailed prompt.
5. Generate the video.
6. Refine and export.

## Basic Prompt Formula

Subject + Action + Environment + Camera + Lighting + Style + Quality

## Example Prompt

Subject: Superhero standing on a rooftop

Action: Looks at the city before jumping into action

Environment: Futuristic cyberpunk city at night

Camera: Slow cinematic dolly-in

Lighting: Neon lights with volumetric fog

Style: Hollywood blockbuster

Quality: Ultra realistic, 8K, highly detailed

## Tips

- Keep prompts specific.
- Use cinematic camera movements.
- Maintain character consistency.
- Describe lighting clearly.
- Generate multiple variations for the best result.
