# 🚀 Getting Started

Follow these steps to get the most out of this repository:

1. Open the tool you want to learn.
2. Read **Beginner.md** first.
3. Continue with **Advanced.md**.
4. Study the remaining guides in order.
5. Practice using the prompt examples.
6. Save your best prompts.
7. Apply what you learn to real projects.

---

# 💡 Best Practices

- Write clear and detailed prompts.
- Be specific about the subject and action.
- Describe the environment.
- Include camera movement when creating videos.
- Specify lighting and mood.
- Choose the appropriate art style.
- Generate multiple variations.
- Refine prompts step by step.
- Build your own prompt library.
