# Complete Prompt Formula

Subject → Details → Style → Typography → Lighting → Color Palette → Composition → Quality

### Example

A premium coffee brand logo with elegant gold typography, minimalist design, black background, soft studio lighting, luxury branding, vector style, ultra high quality.

---

# Professional Workflow

1. Define your project goal.
2. Write a detailed prompt.
3. Choose the best art style.
4. Specify typography requirements.
5. Add lighting and color palette.
6. Generate multiple variations.
7. Compare the results.
8. Refine the prompt.
9. Export the best image.

---

# Best Practices

- Write clear and detailed prompts.
- Use simple, readable typography.
- Match the style to your project.
- Include lighting and color preferences.
- Generate several variations before selecting the final result.

---

# Common Mistakes

- Very short prompts.
- Unclear typography instructions.
- Mixing too many styles.
- Ignoring composition.
- Forgetting quality settings.

---

# Pro Tips

- Build a personal prompt library.
- Save successful prompts.
- Test different aspect ratios.
- Experiment with color palettes.
- Refine prompts gradually for better results.

---

# Conclusion

With strong prompts, thoughtful typography, and consistent practice, Ideogram can produce professional-quality AI images for branding, marketing, content creation, and creative design projects.
