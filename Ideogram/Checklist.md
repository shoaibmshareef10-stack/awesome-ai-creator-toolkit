# Ideogram Checklist

## Before Generating

- Clear subject
- Detailed description
- Appropriate art style
- Correct typography instructions
- Suitable color palette
- Proper lighting
- High quality requested

---

# Prompt Formula

Subject → Details → Style → Typography → Lighting → Colors → Composition → Quality

---

## During Generation

- Check text accuracy
- Verify image composition
- Ensure consistent style
- Review lighting and shadows
- Generate multiple variations

---

## After Generation

- Check spelling in text
- Improve prompt if needed
- Refine colors and composition
- Export the best version

---

# Common Mistakes

- Very short prompts
- Unclear typography
- Conflicting styles
- Poor color choices
- Missing quality instructions

---

# Pro Tips

- Use descriptive prompts.
- Keep typography simple.
- Mention color preferences.
- Generate multiple versions.
- Save your best prompts for future projects.
