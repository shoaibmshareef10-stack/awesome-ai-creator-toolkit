# ⚙️ Installation

## Overview

Before using Firebase services, you need to install the Firebase SDK and connect it to your application. Proper installation ensures your app can communicate securely with Firebase.

---

## Requirements

Make sure you have:

- Node.js (latest LTS version recommended)
- npm or another package manager
- A Firebase project
- A code editor
- A Google account

---

## Install Firebase SDK

Install the Firebase JavaScript SDK using npm:

```bash
npm install firebase
```

Or using Yarn:

```bash
yarn add firebase
```

Or using pnpm:

```bash
pnpm add firebase
```

---

## Verify Installation

Confirm the package is installed:

```bash
npm list firebase
```

---

## Connect Your Application

After installation:

1. Create a Firebase project.
2. Register your app.
3. Copy the Firebase configuration.
4. Initialize Firebase in your project.
5. Connect the required Firebase services.

---

## Recommended Development Tools

- Visual Studio Code
- Git
- GitHub
- Node.js
- Firebase CLI

---

## Installation Tips

- Use the latest Firebase SDK.
- Keep dependencies updated.
- Store configuration securely.
- Test the connection before development.

---

## Common Issues

- Node.js not installed.
- Incorrect project configuration.
- Missing Firebase SDK.
- Invalid Firebase configuration.
- Internet connectivity issues.

---

## Summary

Installing the Firebase SDK is the first step toward building secure and scalable applications using Firebase's cloud services.
