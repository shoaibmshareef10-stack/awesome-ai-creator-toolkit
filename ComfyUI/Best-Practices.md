# ComfyUI Best Practices

## Introduction

Following best practices helps you build efficient, organized, and reusable ComfyUI workflows while generating high-quality AI images consistently.

---

# Workflow Planning

- Define your goal before creating nodes.
- Start with a simple workflow.
- Add advanced features gradually.
- Keep workflows modular.

---

# Node Organization

- Group related nodes together.
- Label important nodes.
- Use reroute nodes to reduce clutter.
- Remove unused nodes regularly.

---

# Model Management

- Use compatible checkpoints and VAEs.
- Organize models into folders.
- Keep LoRAs and ControlNet models updated.
- Delete unused models to save storage.

---

# Performance Tips

- Generate previews before final renders.
- Avoid unnecessary nodes.
- Reuse workflow templates.
- Save stable workflow versions.

---

# Prompt Writing

- Write clear prompts.
- Use effective negative prompts.
- Keep styles consistent.
- Match lighting with the scene.

---

# Project Organization

- Save workflows with descriptive names.
- Create folders for different projects.
- Backup important workflows regularly.
- Keep related assets together.

---

# Common Mistakes

- Overcomplicated workflows.
- Incorrect node connections.
- Using incompatible models.
- Forgetting to save workflows.

---

# Pro Tips

- Build a personal workflow library.
- Test changes one at a time.
- Learn keyboard shortcuts.
- Update ComfyUI and extensions regularly.

---

# Final Checklist

- [ ] Workflow is organized.
- [ ] Nodes are connected correctly.
- [ ] Models are compatible.
- [ ] Prompt is complete.
- [ ] Output settings are correct.
- [ ] Workflow has been saved.
