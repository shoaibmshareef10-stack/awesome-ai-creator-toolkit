
# ComfyUI Beginner Guide

## Introduction

ComfyUI is a powerful node-based interface for AI image generation. It allows you to build custom workflows for creating, editing, and enhancing images with maximum flexibility.

---

# What is ComfyUI?

ComfyUI uses connected nodes instead of a simple prompt box. Each node performs a specific task such as loading a model, encoding text, generating images, or saving results.

---

# Basic Workflow

Load Checkpoint → Text Prompt → Negative Prompt → Sampler → VAE Decode → Save Image

---

# Main Components

- Checkpoint Loader
- CLIP Text Encode
- Empty Latent Image
- KSampler
- VAE Decode
- Save Image

---

# Example Prompt

A futuristic city at sunset, cinematic lighting, ultra realistic, highly detailed, 8K.

---

# Benefits

- Flexible workflows
- Better control over image generation
- Supports advanced models
- Easy workflow sharing
- Professional-quality results

---

# Tips

- Keep workflows organized.
- Name important nodes.
- Save successful workflows.
- Start with simple graphs before creating complex ones.

---

# Common Mistakes

- Connecting nodes incorrectly.
- Using incompatible models.
- Forgetting to save workflows.
- Overcomplicating the graph.
