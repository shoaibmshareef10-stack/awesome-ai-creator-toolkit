# ⌨️ Keyboard Shortcuts

## Overview

Keyboard shortcuts help you work faster by reducing the need to switch between the keyboard and mouse. Learning the most common shortcuts can significantly improve your productivity in Windsurf.

---

## Essential Shortcuts

| Action | Shortcut |
|---------|----------|
| Command Palette | Ctrl + Shift + P |
| Quick Open | Ctrl + P |
| Search in Project | Ctrl + Shift + F |
| Open AI Chat | Ctrl + L |
| Save File | Ctrl + S |
| Find | Ctrl + F |
| Replace | Ctrl + H |
| Terminal | Ctrl + ` |
| Settings | Ctrl + , |

---

## AI Shortcuts

Use shortcuts to:

- Open AI Chat
- Ask questions about code
- Generate code
- Explain selected code
- Fix errors quickly

---

## Navigation Tips

- Use Quick Open to jump between files.
- Use Search to locate functions instantly.
- Learn Command Palette for quick access to all features.
- Memorize the shortcuts you use most often.

---

## Best Practices

- Practice a few shortcuts every day.
- Use shortcuts instead of menus whenever possible.
- Customize shortcuts if supported.
- Focus on the shortcuts that match your workflow.

---

## Summary

Keyboard shortcuts help you navigate faster, write code more efficiently, and make better use of Windsurf's AI-powered features.
