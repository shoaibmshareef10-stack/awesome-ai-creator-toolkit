# 💻 Installation

## Overview

This guide explains how to install Windsurf on Windows, macOS, and Linux.

---

## System Requirements

- Windows 10 or later
- macOS 12 or later
- Linux (64-bit)
- Stable internet connection
- Minimum 4 GB RAM (8 GB recommended)

---

## Download Windsurf

1. Visit the official Windsurf website.
2. Download the installer for your operating system.
3. Run the installer.

---

## Windows Installation

1. Open the downloaded installer.
2. Follow the installation wizard.
3. Launch Windsurf.
4. Sign in to your account.

---

## macOS Installation

1. Open the downloaded DMG file.
2. Drag Windsurf into the Applications folder.
3. Launch the application.
4. Sign in.

---

## Linux Installation

1. Download the Linux package.
2. Install using your package manager.
3. Launch Windsurf.
4. Sign in.

---

## Verify Installation

After installation:

- Open Windsurf.
- Create or open a project.
- Check that AI features are available.
- Confirm that Git integration is working.

---

## Troubleshooting

- Restart the application if it doesn't launch.
- Check your internet connection.
- Update to the latest version.
- Reinstall if necessary.
